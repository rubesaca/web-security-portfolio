# Web Security Portfolio

This repository contains a collection of **real-world web application security findings** identified during independent security research and learning activities.

All vulnerabilities documented here were:
- Found manually
- Verified with minimal, non-intrusive proof of concept
- Reported responsibly to the affected organizations
- Fully anonymized for ethical and legal reasons
